# drevoroda
На данный момент проект не работоспособен и находится на стадии разработки!!!

ЦЕЛЬ:
Создание приложения для формирования индивидуально родословной в удобном для восприятия и анализа виде.

ЗАДАЧА:
Приложение должно содержать следующие функциональные блоки:
- Ввод индивидуальных данных в личном кабинете пользователя;
- Вывод индивидуальных данных в удобнов для восприятия виде;
- Работа с индивидуальными данными возможна только для авторизованных пользователей
Возможны дополниьтельные функции приложения. Будут определяться в дальнейших этапах

РЕШЕНИЕ:
- Создать централизованное полнофункциональное веб-приложение проекта
